## 📅 Day-by-Day Plan

### 📌 Day 1 (วันนี้): Preparation

| เวลา | Task | รายละเอียด |
|------|------|------------|
| **เช้า** | คุยกับแฟน | ขอ Contact คนรู้จักที่ทำ Hostel/ที่พัก |
| **บ่าย** | ทำ Contact List | รวบรวมชื่อ + เบอร์โทร + Facebook |
| **เย็น** | เตรียม Script | ร่างคำพูดสำหรับโทร/ส่งข้อความ |

#### 📝 Template ข้อความแรก (ส่งทาง LINE/Facebook):

```
สวัสดีครับ/ค่ะ คุณ[ชื่อ]

ผม/ฉัน[ชื่อคุณ] เป็นคนรู้จักของ[ชื่อแฟน]ครับ/ค่ะ

ตอนนี้กำลังพัฒนาระบบจัดการที่พักแบบไม่ต้องมี Front Desk 
(ลูกค้าจองเอง + เช็คอินเองด้วยรหัส) 

อยากขอเวลา 15-20 นาที โทรคุยหรือนัดเจอสั้นๆ 
เพื่อเข้าใจปัญหาของเจ้าของที่พักจริงๆ ครับ/ค่ะ

(ไม่ได้จะขายอะไร แค่อยากฟัง Feedback ครับ/ค่ะ)

สะดวกวันไหนบ้างครับ/ค่ะ?
```

### 📌 Day 2: First Outreach

| Task | จำนวน | วิธี |
|------|-------|------|
| ส่งข้อความ | 5-10 คน | LINE / Facebook Messenger |

### 📌 Day 3-4: Problem Interviews

| Task | รายละเอียด |
|------|------------|
| นัดโทร/Video Call | กับคนที่ตอบรับ |
| ใช้ Interview Script | (ดูด้านล่าง) |
| จดบันทึก | ทุกคำตอบสำคัญ |

#### 📋 Interview Script (15-20 นาที):

```markdown
## เปิด (2 นาที)
"ขอบคุณที่สละเวลาครับ ผมอยากเข้าใจการทำที่พักให้มากขึ้น 
ไม่มีคำตอบถูกผิด แค่อยากฟังประสบการณ์จริงครับ"

## ถามเรื่องปัจจุบัน (5 นาที)
1. "ที่พักมีกี่ห้องครับ? เปิดมานานแค่ไหน?"
2. "ตอนนี้จัดการ Booking ยังไงครับ? ใช้ Software อะไร?"
3. "มีพนักงานกี่คน? ทำหน้าที่อะไรบ้าง?"

## ถามเรื่องปัญหา (5 นาที)
4. "อะไรคือปัญหาใหญ่สุดในการจัดการที่พักตอนนี้ครับ?"
5. "เคยมีปัญหาเรื่องพนักงาน Front Desk ไหมครับ? เช่นอะไร?"
6. "ถ้าลดพนักงานได้ 1 คน จะประหยัดเดือนละเท่าไหร่?"

## ถามเรื่อง Solution (5 นาที)
7. "ถ้ามีระบบที่ลูกค้าจองเอง + เช็คอินเองด้วยรหัส จะสนใจไหม?"
8. "ถ้าระบบนี้ราคา 1,500 บาท/เดือน รู้สึกยังไง?"
9. "อะไรจะทำให้ตัดสินใจใช้? อะไรจะทำให้ไม่ใช้?"

## ปิด (3 นาที)
10. "ถ้าผมทำระบบนี้ออกมา แล้วให้ลองใช้ฟรี 3 เดือน 
    แลกกับ Feedback ทุกสัปดาห์ จะสนใจไหมครับ?"

"ขอบคุณมากครับ Feedback มีค่ามากครับ"
```

|-------|---------|---------|---------|
| จำนวนห้อง | | | |
| ปัญหาใหญ่สุด | | | |
| ต้นทุนพนักงาน/เดือน | | | |

### 📌 Day 5: Analysis & Pivot

| Task | รายละเอียด |
|------|------------|
| สรุปผล Interview | ดูว่าคนตอบ Yes กี่คน |
| หา Pattern | Pain Points ที่ซ้ำกัน |
| ตัดสินใจ | ไปต่อ / Pivot / หยุด |

#### 🚦 Decision Framework:

| ผลลัพธ์ | Action |
|---------|--------|
| **3/5 สนใจ Pilot** | ✅ ไปต่อ! เริ่มเขียนโค้ด |
| **1-2/5 สนใจ** | 🟡 หาคนอื่นเพิ่ม หรือปรับ Value Proposition |
| **0/5 สนใจ** | 🔴 Pivot: เปลี่ยนกลุ่มเป้าหมาย หรือหยุดโปรเจค |

### 📌 Day 6-7: Pilot Setup

**ถ้าได้ Pilot Customer:**

| Task | รายละเอียด |
|------|------------|
| ยืนยันข้อตกลง | ใช้ฟรี 3 เดือน / Feedback ทุกสัปดาห์ |
| เก็บข้อมูลที่พัก | ชื่อ, ที่อยู่, จำนวนห้อง, ราคา |
| กำหนด Kickoff | นัดวันเริ่มต้นใช้งาน |
| เริ่ม Phase 1 | Setup Project + เริ่ม Coding! |

#### 📝 Pilot Agreement (ง่ายๆ):

```markdown
## ข้อตกลง Pilot Program

ระหว่าง: [ชื่อคุณ] และ [ชื่อที่พัก]

1. [ชื่อที่พัก] จะได้ใช้ระบบ StayLock ฟรี 3 เดือน
2. แลกกับ:
   - Feedback ทุกสัปดาห์ (โทร 15 นาที หรือ LINE)
   - อนุญาตให้ใช้เป็น Case Study (ถ้าพอใจระบบ)
3. หลังจบ 3 เดือน:
   - ถ้าสมัครต่อ: ราคาพิเศษ 1,000 บาท/เดือน ตลอด
   - ถ้าไม่สมัคร: ยกเลิกได้เลย ไม่มีค่าใช้จ่าย

ลงชื่อ: _____________ วันที่: _____________
```

## 📞 Backup Plan: Cold Outreach

**ถ้า Connection ไม่พอ:**

### หา Hostel จาก Google Maps:

1. เปิด Google Maps
2. Search: "hostel [จังหวัดเป้าหมาย]"
3. เลือก Hostel ที่มีรีวิว 3-4 ดาว (ไม่ใหญ่มาก ไม่เล็กมาก)
4. หา Facebook Page
5. ส่ง DM ด้วย Script นี้:

```
สวัสดีครับ/ค่ะ ทีมงาน[ชื่อ Hostel]

ผม/ฉันกำลังพัฒนาระบบจัดการที่พักสำหรับ Hostel ขนาดเล็กในไทย
และกำลังหา Pilot Partner มาช่วยทดสอบระบบ (ใช้ฟรี 3 เดือน)

ระบบช่วยให้:
✅ ลูกค้าจองและจ่ายเงินเองผ่านเว็บ
✅ ส่งรหัสเช็คอินอัตโนมัติ ไม่ต้องมีคนรอ
✅ จัดการห้องง่ายๆ ผ่าน Dashboard

ถ้าเจ้าของสนใจ ขอเบอร์โทรติดต่อได้ไหมครับ/ค่ะ?
ขอบคุณครับ/ค่ะ 🙏
```

## 🚨 Common Mistakes to Avoid

| ❌ ห้ามทำ | ✅ ทำแทน |
|----------|----------|
| รีบเขียนโค้ดก่อนคุยลูกค้า | คุยลูกค้าก่อน 5 คน แล้วค่อยเริ่ม |
| ขายของตอน Interview | ฟังอย่างเดียว ถามคำถาม |
| สมมติว่ารู้ปัญหา | ถามจริงๆ ว่าปัญหาคืออะไร |
| ถามว่า "ดีไหม?" | ถามว่า "จะจ่ายเท่าไหร่?" |
| เชื่อคำพูด "น่าสนใจนะ" | เชื่อเมื่อเซ็นเป็น Pilot |
